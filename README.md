# Sp Ayam Broiler

# Requirements

- PHP 8.2 or higher (Xampp 8.2)
- Composer
- Node.js & npm

# Instalasi

```bash
git clone https://github.com/lukheman/sp-ayam-broiler.git
composer install
php artisan migrate
php artisan migrate:fresh --seed
```

# Penggunaan

```bash
php artisan serve
```

Akses di `http://localhost:8000`
